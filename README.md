# Wprowadzenie do analizy danych i uczenia maszynowego

Repozytorium zawiera materiały do projektu edukacyjnego "Wprowadzenie do analizy danych i uczenia maszynowego" organizowanego przez Koło Naukowe Data Science w semestrze zimowym roku akademickiego 2025/2026.

Celem projektu jest wprowadzenie nowych członków Koła Naukowego Data Science w świat analizy danych i uczenia maszynowego. Projekt zostanie podzielony na etapy, przy czym każdy etap będzie składał się z wykładu łączącego teorię z praktyką i z zadania do wykonania indywidualnie przez każdego z uczestników. Dodatkowo uczestnicy wykonają prosty projekt w zespołach 2 lub 3 osobowych, który przedstawią na spotkaniu podsumowującym projekt.
